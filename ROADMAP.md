# ROADMAP.md

## MVP 0.1 — Base do projeto

Objetivo:
- estruturar o repositório
- definir arquitetura
- preparar package layout com `src/`
- padronizar schema universal

Entregas:
- arquivos de contexto
- estrutura de diretórios
- `pyproject.toml`
- `requirements.txt`
- parsers e pipeline mínimos

## MVP 0.2 — Pipeline funcional

Objetivo:
- processar arquivo único e pasta
- gerar artefatos padronizados

Entregas:
- roteamento por extensão
- parsers básicos
- extração de metadados
- índice estrutural
- resumo inicial
- escrita em JSON e Markdown

## MVP 0.3 — Robustez operacional

Objetivo:
- melhorar qualidade dos parsers
- lidar com arquivos problemáticos
- criar logging e testes melhores

Entregas:
- tratamento de erro por parser
- logs consistentes
- testes automatizados
- normalização mais estável

## MVP 0.4 — Catálogo e lote

Objetivo:
- consolidar uma pasta inteira em visão de conjunto

Entregas:
- índice mestre da pasta
- catálogo geral de arquivos
- visão resumida do acervo
- metadados agregados

## MVP 0.5 — Busca e RAG

Objetivo:
- preparar consulta inteligente sobre os arquivos

Entregas:
- busca textual
- chunking semântico
- embeddings
- base para RAG local
